Restore Instructions

← Backups

Warning

Restoring data will modify or overwrite your production database. Always test restores on a staging copy first and keep an extra backup.

Restore using psql (Postgres)

-- Example: restore users.csv into `users` table
psql "$DATABASE_URL" -c "\copy users(id,username,email,is_verified,created_at) FROM 'backups/2026-06-01/users.csv' CSV HEADER;"

-- For other tables, adapt columns accordingly.

Restore using Python (recommended for complex imports)

Example outline:

from models import db, User
import csv

with open('backups/2026-06-01/users.csv', newline='', encoding='utf-8') as fh:
    reader = csv.DictReader(fh)
    for row in reader:
        user = User(id=row['id'], username=row['username'], email=row['email'], is_verified=row['is_verified'])
        db.session.merge(user)  # merge will insert or update based on PK
    db.session.commit()
